================================================================================
WORKFLOW & PATTERNS
================================================================================

1. CONTENT MANAGEMENT:
   ┌─────────────────────────────────────────┐
   │ WordPress Admin Editor                   │
   │ ├─ Post Meta Fields (WPAlchemy Metabox) │
   │ ├─ Settings Pages (Theme Options)        │
   │ └─ Custom Post Types (Photos, etc.)      │
   └────────────┬────────────────────────────┘
                │
                ↓
   ┌─────────────────────────────────────────┐
   │ Template Files fetch data via:           │
   │ ├─ get_post_meta()                       │
   │ ├─ get_option()                          │
   │ ├─ WP_Query                              │
   │ └─ get_posts()                           │
   └────────────┬────────────────────────────┘
                │
                ↓
   ┌─────────────────────────────────────────┐
   │ Frontend Display                         │
   │ (Using Bootstrap 5 + Custom CSS)         │
   └─────────────────────────────────────────┘

2. PAGE TEMPLATE HIERARCHY:
   Custom Templates (about.php, pricing-page.php, contact-page.php)
                        ↓
   Specific Archives (single.php, category.php, taxonomy-photoshoots.php)
                        ↓
   Generic Fallbacks (taxonomy.php, index.php)

3. COMPONENT REUSABILITY:
   Templates loaded via: get_template_part('templates/component-name', null, $args)
   
   Example: get_template_part('templates/testimonials')
   This loads: templates/testimonials.php with inherited scope

4. DATA FLOW:
   WordPress Data
   (Posts, Meta, Options)
        │
        ↓
   Template Variables
   (PHP $variables)
        │
        ↓
   HTML Output
   (Bootstrap classes + custom CSS)
        │
        ↓
   JavaScript Enhancement
   (jQuery, WOW.js, Owl Carousel, etc.)

KEY DEPENDENCIES & TECHNOLOGIES
================================================================================

📦 BUILD TOOLS:
   • Webpack 5          - Module bundler
   • SASS/SCSS          - CSS preprocessing
   • PostCSS            - CSS compatibility (autoprefixer)
   • PurgeCSS           - Remove unused CSS
   • Terser             - JS minification
   • Mini CSS Extract   - Extract CSS to separate files

📦 FRONTEND FRAMEWORKS:
   • Bootstrap 5.1.3    - CSS framework
   • jQuery 3.6.1       - DOM manipulation
   • Swiper/Carousel    - Slider components
   • Masonry Layout     - Grid layout
   • Owl Carousel 2.3.4 - Product carousels

📦 ANIMATIONS & INTERACTIONS:
   • WOW.js 1.1.3       - Scroll animations
   • Magnific Popup 1.2 - Lightbox
   • Imagesloaded 5.0   - Image loading detection

📦 WORDPRESS PLUGINS:
   • WPAlchemy          - Metabox framework
   • TinyMCE Shortcodes - Editor integrations

📦 INTEGRATIONS:
   • Google reCAPTCHA   - Form protection
   • Google Analytics   - GA ID: G-WP6KF2ZDH2
   • Google Fonts       - FontAwesome 5.8.1
   • TrustIndex         - Airbnb/review embeds

================================================================================
BUILD PROCESS
================================================================================

SOURCE FILES (assets/)
   │
   ├─ SCSS Compilation (main.scss + components)
   ├─ JavaScript Bundling (scripts.js + vendors)
   │
   ↓
   
WEBPACK Processing
   │
   ├─ PurgeCSS (remove unused Bootstrap + CSS)
   ├─ Autoprefixer (add vendor prefixes)
   ├─ Minification (CSS & JS)
   │
   ↓
   
DIST FILES (production-ready)
   │
   └─ Served to frontend via functions.php/wp_enqueue_*

Commands:
   npm start   → Development mode (watch for changes)
   npm build   → Production build (optimized, minified)
